# NanoPulse

> **Note:** This repository is under active development on the `dev` branch.
>
> The `master` branch will remain empty until the first stable release (v1.0.0).

## About

NanoPulse is a Nextflow DSL2 pipeline for de novo clustering and consensus building for Oxford Nanopore Technologies (ONT) 16S rRNA sequencing data.

This pipeline is based on **[NanoCLUST](https://github.com/genomicsITER/NanoCLUST)** by Héctor Rodríguez-Pérez, Laura Ciuffreda, and Carlos Flores.

## Development

Active development is happening on the **`dev`** branch:

\`\`\`bash
git clone https://github.com/FOI-Bioinformatics/NanoPulse.git
git checkout dev
\`\`\`
